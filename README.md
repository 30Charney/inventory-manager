# Inventory Manager

Sistema de gerenciamento de estoque desenvolvido em Python e SQLite.

## Funcionalidades

- Cadastro de produtos
- Consulta de produtos
- Atualização de estoque
- Remoção de produtos

## Tecnologias

- Python
- SQLite
- SQL
- GitHub
